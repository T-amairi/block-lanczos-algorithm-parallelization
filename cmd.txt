- Grid5000 :
    >> MPI/Hybrid :
        * (Ethernet network) oarsub -p "cluster='cluster_name'" -l nodes=node_number -I
        * (Infiniband network) oarsub -p "cluster='cluster_name'" -p "ib_rate=56" -l nodes=node_number -I 
        * (Omni-Path network) oarsub -p "cluster='cluster_name'" -p "opa_rate=100" -l nodes=node_number -I

    >> OpenMP : 
        * oarsub -l host=1/core=core_number -I

    >> Examples :
        * Ethernet network : site = Nancy, cluster = gros
        * Infiniband network : site = Nancy, cluster = grcinq
        * Omni-Path network : site = Grenoble, cluster = dahu

- lanczos_modp : 
    >> --matrix FILENAME           MatrixMarket file containing the sparse matrix
	>> --prime P                   compute modulo P
	>> --n N                       blocking factor [default 1]
	>> --output-file FILENAME      store the block of kernel vectors
	>> --right                     compute right kernel vectors
	>> --left                      compute left kernel vectors [default]
	>> --stop-after N              stop the algorithm after N iterations
	>> --mpi-block-product         enable mpi implementation of the block dot product
	>> --mpi-ortho           	   enable mpi implementation of the orthogonalization phase
	>> --checkpoint cp             enable checkpointing every cp seconds [default cp = 60 s]
	>> --load-checkpoint           load vectors from checkpointing files 
	>> The --matrix and --prime arguments are required
	>> The --stop-after and --output-file arguments mutually exclusive

- MPI :
    >> (Ethernet/Infiniband network) mpiexec -machinefile $OAR_NODEFILE --map-by ppr:1:core ./lanczos_modp [options]
    >> (Omni-Path network) mpiexec -machinefile $OAR_NODEFILE --map-by ppr:1:core -mca mtl psm2 -mca pml ^ucx,ofi -mca btl ^ofi,openib ./lanczos_modp [options] 

- OpenMP : (export OMP_STACKSIZE="1G" on root node first)
    >> ./lanczos_modp [options]

- Hybrid : (export OMP_STACKSIZE="1G" on root node first)
    >> (Ethernet/Infiniband network) mpiexec -x OMP_STACKSIZE="1G" -machinefile $OAR_NODEFILE --map-by ppr:1:node ./lanczos_modp [options]
    >> (Omni-Path network) mpiexec -x OMP_STACKSIZE="1G" -machinefile $OAR_NODEFILE --map-by ppr:1:node -mca mtl psm2 -mca pml ^ucx,ofi -mca btl ^ofi,openib ./lanczos_modp [options]
